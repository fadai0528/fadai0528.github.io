# Exam Assistant

嶺東高中國中部段考助手是一個純前端靜態頁面工具，用來在考場顯示考程、時間、班級、叮嚀事項與座位狀態。

## 頁面

- `new-exam.html`：目前主要版本，包含高 DPI 模式、深色模式、全螢幕、系統設定、座位表與考程顯示。
- `ea.html`：舊版或備用頁面。

直接用瀏覽器開啟 `new-exam.html` 即可使用，不需要後端服務。

## 主要功能

- 顯示目前日期、時間與考程狀態。
- 可設定考試名稱、班級、考場叮嚀、考程與座位表。
- 可匯出與匯入目前設定 JSON 檔。
- 座位可標記為在場、缺席或抽離。
- 支援深色模式與全螢幕顯示。
- 支援高 DPI 模式，改善高解析度或高縮放設備上的座位表溢出問題。
- 設定會保存在瀏覽器 `localStorage`。

## 本地資料

`new-exam.html` 使用以下 `localStorage` key 保存設定：

```text
LingTung_v24_M3_Layouts
```

目前頁面只讀寫這個 key，不會清空整個 `localStorage`，也不會主動刪除既有資料。若這個 key 中的 JSON 損壞，頁面會改用預設資料載入，並保留原始損壞資料。

## 高 DPI 模式

頁首的「高 DPI 模式」按鈕會切換較緊湊的版面：

- 降低面板間距與座位格間距。
- 縮小座位文字與狀態標籤。
- 讓座位表用可收縮的 grid 列高與欄寬，避免超出右側座位面板。

切換狀態會保存在同一份 `localStorage` 設定中。

## 考程輸入

在系統設定中新增或編輯考科時，每列包含：

- 開始時間
- 結束時間
- 科目

時間欄位可直接輸入數字，不需要手動輸入 `:` 或 `-`：

- `835` 會轉成 `08:35`
- `0835` 會轉成 `08:35`
- `925` 會轉成 `09:25`
- `0925` 會轉成 `09:25`

儲存後仍會使用原本的資料格式，例如：

```json
{ "t": "08:35 - 09:25", "s": "國文" }
```

## 匯入與匯出設定

在系統設定底部點擊「匯出設定」會下載 JSON 檔，檔名格式如下：

```text
exam-settings-YYYYMMDD-HHMM.json
```

匯出內容包含版本、匯出時間與目前設定資料。匯出不會清除或修改瀏覽器本地資料。

點擊「匯入設定」可選擇 JSON 檔並立即套用。支援兩種格式：

```json
{
  "version": 1,
  "exportedAt": "2026-06-25T10:00:00.000Z",
  "data": {
    "school": "考試名稱",
    "className": "班級"
  }
}
```

也支援直接匯入設定物件：

```json
{
  "school": "考試名稱",
  "className": "班級"
}
```

匯入成功後會寫回同一個 `localStorage` key 並立即更新畫面。若 JSON 格式錯誤，頁面會提示匯入失敗，並保留目前資料。

## 開發與測試

安裝依賴：

```bash
npm install
```

執行測試：

```bash
npm test
```

測試使用 Playwright，覆蓋：

- 高 DPI 模式下座位表不溢出。
- 既有 `localStorage` 資料不會消失。
- 損壞的 `localStorage` JSON 不會讓頁面掛掉。
- 使用者輸入的 HTML-like 字串會以文字顯示，不會被注入成 DOM。
- 非法座位網格尺寸會被限制在合理範圍。
- 時鐘不會重複啟動多個計時器。
- 設定可匯出與匯入 JSON。
- 考程開始/結束時間可用純數字輸入並自動格式化。

## 注意事項

- 座位網格行列目前限制為 `1` 到 `12`。
- 若修改資料結構，請維持對既有 `LingTung_v24_M3_Layouts` 資料的相容性。
- 不要使用 `localStorage.clear()` 或刪除整個瀏覽器本地資料。
